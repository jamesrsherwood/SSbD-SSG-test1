# Safe and Sustainable by Design (SSbD) Solvent Selection Guide (SSG)
The [SSbD SSG](https://jamesrsherwood.github.io/SSG) is a tool for synthetic organic chemistry.

## Features
- User interface to define reactions and process conditions.
- Predicted yields, and climate change and freshwater use impacts.
- A database to look up solvent properties.
- Safe and Sustainability by Design analysis for 365 solvents.
- A bespoke solvent selection guide based on user input.

## Navigation
Choose from preset reactions and autocomplete reagents on the [user interface](https://jamesrsherwood.github.io/SSG/database.html). Masses contribute to Green Chemistry metrics. Conversions are calculated based on solvent effects specific to each reaction. The process and protective measures determines exposure. Standardised risk scenario options can be selected to calculate risk.
